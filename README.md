## React native infinite scroll using FlatList

## Clone this repo
```
git clone https://github.com/samironbarai/rn-infinite-scroll.git
```

## Install dependency
```
$ cd rn-infinite-scroll
$ npm install
```

## Run the project
```
react-native run-android
```
